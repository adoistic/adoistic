### Open source

- **[Falsafa](https://github.com/adoistic/falsafa).** A premium reading platform for translated philosophical and classical texts, with an open-source MCP engine any LLM can query the corpus through. Karpathy-flavored: tools, not vectors.
- **[forme](https://github.com/adoistic/forme).** An editorial magazine CMS for print. Single operator, offline. Open source.
- **[mineru-true-copy](https://github.com/adoistic/mineru-true-copy).** A Tauri desktop app that turns PDFs into editable documents while keeping the page intact. Built on MinerU, with vision-LLM OCR that preserves bold and italics, layout-preserving export to DOCX/PPTX/PDF, and an Indic translation sidecar for Apple Silicon. AGPL-3.0.
- **[pdf-toolkit](https://github.com/adoistic/pdf-toolkit).** PDF tools for cultural archives.
